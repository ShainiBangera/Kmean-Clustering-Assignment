**This repository consists of KMean clustering assignment which groups countries based on socio-economic data.**

**Dataset used:**
Unsupervised Learning on Country Data: "Country-data.csv"
https://www.kaggle.com/datasets/rohan0301/unsupervised-learning-on-country-data?select=Country-data.csv

**Steps Performed:**
- Data Preprocessing
- Feature Scaling using  StandardScaler
- Used Elbow Method to choose K
- Applied K-Means clustering
- Visualized country clusters

**Results:**
Countries are grouped into 3 clusters based on the data.

**Visualization:**
-Scatter plot of 'child_mort' vs 'exports'.
-Scatter plot of 'health' vs 'total_fer'.
